# Professional Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Task Objectives

This project aims to create a professional portfolio website that showcases the following:

- About information and personal introduction
- Technical and non-technical skills
- Projects with descriptions and links
- Contact form for visitors
- Responsive design for mobile compatibility

## Features

- **Responsive Design**: Fully responsive layout that works on all device sizes
- **Navigation**: Smooth scrolling navigation with mobile-friendly menu
- **Hero Section**: Introduction with call-to-action buttons
- **About Section**: Personal information with education and work experience
- **Skills Section**: Visual representation of technical skills and other competencies
- **Projects Section**: Showcase of projects with descriptions, technologies used, and links
- **Certificates Section**: Display of professional certifications and skills acquired
- **Contact Form**: Form for visitors to send messages with validation
- **Social Links**: Links to professional profiles like LinkedIn and GitHub

## Steps to Run the Project

1. **Clone the Repository**
   \`\`\`
   git clone https://github.com/Tanvir1610/portfolio-website.git
   cd portfolio-website
   \`\`\`

2. **Open the Project**
   - Simply open the `index.html` file in any modern web browser

3. **For Development**
   - To make changes, edit the HTML, CSS, and JavaScript files as needed
   - No build tools or dependencies are required

## Customization

To personalize this portfolio:

1. Replace placeholder content in `index.html` with your own information
2. Update skills and their percentages in both HTML and CSS
3. Add your own projects with descriptions and links
4. Replace placeholder images with your own pictures
5. Update contact information and social media links
